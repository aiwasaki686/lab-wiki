# 空気敏感試料等の秤量について<br>

## 有害な気体を発する試料について<br>

保護メガネと手袋は必須<br>
秤量はバイアル瓶で行うが、びん本体とふたを両方秤量し、ドラフト内で少しずつ移しながら行っていく<br>
ふた付きで秤量することを絶対忘れない!!<br>
ドラフトを少し下げて行うとなおよいかも<br>
秤量操作は素早く丁寧に(なんでも)<br>

## 潮解性を有する試料について<br>

基本的にガラス器具(ビーカー)を用いて秤量する<br>
ろ紙の場合、試料が溶液となってしまったときに張り付いてしまい、正確に秤量することが困難となる<br>
水酸化物を水溶液とするとき、少しずつ水を加えていく(発熱反応よりビーカー内がとても熱くなる)<br>
潮解性を有する化合物はデシケータ中段で保管している
