# オゾン分解操作マニュアル

> 最終更新：2026-07-02
> 最終編集者：笠井

## 使用器具・装置

* 低温反応器
* オゾン発生器
* 酸素ボンベ
* 窒素ボンベ
* レギュレーター
* ホース
* 綺麗ななパスツール
    * サンプル数分を用意する
* パラフィルム
* ニトリル手袋
* 小バイアル

## 1. 事前準備

### 1.1 低温反応器の冷却

低温反応器の温度を −78 ℃ に設定する。

設定温度に到達するまでに 2〜3時間程度 かかるため、実験開始時刻から逆算して準備する。

この間にサンプルを調製する。
基本的に、サンプル量が数 mg 程度の場合は、小バイアル中で 約1 mL の適切な溶媒に溶解する。
使用する溶媒や濃度は、先行研究を参考にして適宜決定する。

### 1.2 酸素ボンベの準備

窒素ガスを使用する予定の人がいないことを確認してから、レギュレーターを酸素ボンベに付け替える。

また、あらかじめオゾン分解するサンプルを低温反応器に入れ、十分に冷却しておく。

⚠️ 注意
ボンベやレギュレーターの取り扱いは危険を伴うため、操作に不安がある場合は必ず教員または先輩に確認する。

### 1.3 オゾン発生器とホースの接続

1. ドラフト内にオゾン発生器を設置する。
2. オゾン発生器の 入口側 にホースを接続し、酸素ボンベとつなぐ。
3. 接続部をネジでしっかり固定する。
4. オゾン発生器の 出口側 に、パスツールを取り付けるためのホースを接続する。
5. 入口側と同様に、接続部をしっかり固定する。

⚠️ 注意
ホースの固定が不十分だと、酸素やオゾンが漏れる可能性がある。接続部に緩みがないことを必ず確認する。

### 1.4 パスツールの取り付け

1. オゾン発生器の出口側ホースにパスツールを取り付ける。
2. 接続部をパラフィルムでしっかり巻く。

⚠️ 注意
パラフィルムの巻きが不十分だと、接続部からガスが漏れる可能性がある。

### 1.5 オゾン発生器の流量調節弁の確認

オゾン発生器の流量調節弁を右回りに回し、弁が閉じていることを確認する。
この時点では、まだオゾン発生器の電源は入れない。

### 1.6 酸素ボンベの開栓

1. 酸素ボンベの一次弁をゆっくり開ける。
2. 二次弁を調整し、圧力を 0.1 MPa に合わせる。
3. 三次弁を開ける。

⚠️ 注意
一次弁を急に開けたり、開けすぎたりすると、ボンベ内の酸素が急激に流出して危険である。必ずゆっくり操作する。

### 1.7 酸素ガスの流量調整

オゾン発生器の流量調節弁をゆっくり左回りに開き、酸素ガスの流量を 100 mL/min に調整する。
この操作により、オゾン発生器の出口側から酸素ガスが流れる。

### 1.8 ガス圧力の再調整

オゾン発生器のガス圧力が約 0.1 MPa になるように、酸素ボンベの圧力および流量を再調整する。

## 2. オゾン通液

### 2.1 通液前の確認

1. オゾン発生器の電源スイッチを入れる。
2. ニトリル手袋を着用する。
3. サンプルが十分に冷却されていることを確認する。
サンプルは、−78 ℃で3分程度 冷却されていればよい。

### 2.2 オゾン発生条件の設定

出力調節ボリュームを回し、以下の条件に調整する。

項目	設定値
圧力	0.1 MPa
流量	100 mL/min
出力	30
電流	0.2 A

出力を上げると、この時点でオゾンが発生し始める。

### 2.3 オゾンの通液

パスツールをサンプル溶液中に入れ、オゾンを通液する。
反応時間は、サンプルの性質や先行研究を参考にして適宜決定する。

⚠️ 注意
オゾンは有害であるため、操作は必ずドラフト内で行う。
通液中は、ドラフトの扉をできるだけ下げて作業する。

### 2.4 オゾン発生の停止

反応終了後、出力調節ボリュームを 0 に戻し、オゾン発生器の電源を切る。
これにより、発生するガスはオゾンガスから酸素ガスに切り替わる。

⚠️ 注意
この時点では、サンプル溶液中にオゾンが残っている可能性がある。
サンプルを低温反応器から取り出さないこと。

### 2.5 残存オゾンの排出

サンプル溶液中に残ったオゾンを排出するため、酸素ガスを 3分程度 通液する。

### 2.6 酸素ボンベの停止と窒素ボンベへの付け替え

1. 酸素ボンベの一次弁を閉める。
2. ホースを外す。
3. レギュレーター内に残った酸素を出し切る。
4. 三次弁を閉じる。
5. レギュレーターを再び窒素ボンベに付け替える。
6. オゾン発生器に接続していたホースを窒素ラインにつなぐ。


### 2.7 窒素通液

サンプル溶液中に窒素ガスを 3分程度 通液する。

## 3. 片付け

### 3.1 サンプルの後処理

サンプル溶液を濃縮し、必要に応じて乾燥する。

### 3.2 器具の片付け

使用した器具を片付ける。

* ホースを外す
* オゾン発生器を元の位置に戻す
* 使用したバ周辺器具を片付ける

### 3.3 低温反応器の停止

1. 低温反応器の電源を切る。
2. 触れられる程度まで温度が上がったことを確認する。
3. 低温反応器内のビーズを取り出す。


## 4. 全体の注意事項

* オゾンは有害であるため、必ずドラフト内で操作する。
* 実験中はドラフトの扉をできるだけ下げる。
* ニトリル手袋を着用する。
* ホース、パスツール、オゾン発生器の接続部からガス漏れがないことを確認する。
* 低温反応器を使用するため、凍傷に注意する。
