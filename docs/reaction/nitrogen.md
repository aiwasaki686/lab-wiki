# 窒素風船の作り方<br>

1. 風船2個、半透明なチューブ約5 cm、テルモシリンジ、注射針、白いテープ、パラフィルムを用意する<br>
- 風船 -> グローボックスがある実験台の3段目<br>
- 白いテープ -> グローボックスがある実験台の2段目<br>
- 半透明なチューブ -> 中圧がある実験台の下<br>
- テルモシリンジ -> 山田研HPLCがある実験台の左側の一番上の棚<br>
- 注射針 -> テルモシリンジとどうよな場所<br>

2. 風船の中に風船を入れる(中にある風船がなるべく伸びるように)<br>

3. 風船の口に半透明なチューブをはめる<br>

4. 風船の口とチューブが接している部分をパラフィルムで覆う<br>

5. テルモシリンジを0.6 mLくらいで切断する<br>
 切断した部分を白いテープでぐるぐるに巻く<br>
(注射針が刺さる方を使うため、シリンジと切断された側は捨ててしまって構わない)<br>

6. 切断したテルモシリンジと風船をはめたチューブをはめる<br>
緩い場合は白いテープをさらに巻く<br>

7. シリンジとチューブが接している部分をパラフィルムで覆う<br>

8. 注射針をつけることで完成<br>
実際に窒素を入れて正常に膨らむことを確認する<br>

# 窒素置換<br>

1. 窒素ボンベとドラフト内のガラス管をつなぐチューブがつながっていることを確認<br>
窒素置換はドラフト左側にある特殊な形状のガラス管がつながっている2か所のみで行うことができる<br>

2. 窒素ボンベにある一番右の金属の棒を奥に向ける<br>

3. 窒素ボンべの一番左にある黒いねじを一回転半させる<br>

4. 特殊なガラス管の奥にある試験管から気泡が出ていることを確認<br>
出ていない場合、ガラス管の左にあるコックが閉じられていることが考えられる<br>
コックが空いている状態 = 黒い線がそろっている<br>
  
5. フラスコにセプタムを取り付け、窒素風船を刺す<br>

6. フラスコを真空引きする<br>
真空引きをするときは太いつまみが上にくるように回す<br>
風船の様子を確認し、しぼみきっていることが真空引きのサイン<br>
  
7. つまみをゆっくり回し、細いつまみが上にくるように回す<br>
このとき、試験管に刺さっているパスツールピペットの様子を確認し、試験管内の油が吸われすぎないようにゆっくり回す<br>

8. 風船が膨らむこと確認<br>
コックを閉じることで窒素がフラスコのみに行くようになり、窒素の導入が速くなる<br>
風船が全く膨らまない場合は、窒素を吹き込んでいる注射針が詰まっていることが考えられるため、交換するなどして様子を見る<br>

9. 6から8の操作を計3回行い、3回目の窒素封入後は真空引きを行わずそのまま反応に用いる。
