# シリンジの使い方及び片付け

### シリンジの使い方<br>

1. デシケータよりシリンジを出す<br>
針の大きさは18と20の2種類あり、20のほうが太くなっている<br>
懸濁液を用いる場合は太いほうが針先が詰まらないかも<br>

2. 窒素風船を脱水溶媒にさす<br>
風船は窒素によって膨らませるのだが、窒素を入れて出す操作を2回行った後、窒素で満たす<br>
 -> 風船内の空気を窒素に置換するために行う<br>
 脱水ではない普通の試薬の場合は可能な限り空気が入らないようドラフト内で操作を行う<br>

3. 溶媒の瓶に針先を少し入れ、溶媒に触れていない状態でシリンジ内を窒素で満たす<br>

4. 空気中でシリンジ内に満たされた窒素を押し出す<br>

5. 3と4の操作を2回行い、十分にシリンジ内部を窒素で置換したのちに溶媒を吸い取る<br>
必要分を吸い取った後、溶媒から針を抜き溶媒のびんの中でピストンを引くことでシリンジ内に少し窒素が入る<br>
溶液を窒素でふたすることでフラスコに移す最中に溶液が漏れることが減る<br>
溶液をフラスコに移すときはピストンを少しづつ押し、シリンジ内部の窒素を出しながら行うことで反応系中に空気が入ることを減らす<br>

### シリンジの片づけ方<br>

1. 使用後はまず水とアセトンで洗浄する<br>

2. シリンジの先をピンセットを用いて外し、アセトンで全体を流す<br>

3. アセトンをドライヤーで飛ばし、乾燥器内に入れる<br>

4. シリンジ本体をアセトンで流しドライヤーで飛ばす(針の接続部分をよく洗う)<br>

5. 乾燥機に入れるのだが、小さいシリンジはアルミホイルで簡易的な入れ物を作りそこに入れてから乾燥器内に入れる<br>
乾燥器の一番下には小さな穴が開いており、そこに落ちてしまうことがあるため、その防止として行って欲しい<br>

6. 十分に乾燥させた後はかごに入れて冷ますのではなく、すぐにデシケータにしまうこと!!(by 原先輩)
