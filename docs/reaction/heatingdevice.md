# 高温反応器(加熱装置)の取り扱い

**操作一覧**

-電源の付け方と注意点<br>
-回転数の設定<br>
-温度設定<b>

## 電源の付け方と注意点


-加熱装置右置くにスイッチがあるためそこでオンオフを切り替える。

-付属の金属棒を装置の受け部分に差し込む(これをささないと際限なく装置の加熱が行われてしまう)。

## 回転数の設定

-装置左側のダイヤルによって回転数を変えることができる。

-装置のオンオフは▶|□を押す。

-▶|□の右にあるボタンは回転の種類の番号を変えることができる。

## 温度設定

-装置左側のダイヤルによって回転数を変えることができる。

-装置のオンオフは▶|□を押す。

-錠前のボタンを押すと設定温度を固定できる(ダイヤルを回しても温度が動かない)。

# オイルバスの取り扱い

-右のレバーよりオンオフを切り替えられる。

-左のダイヤルにより温度を変えられる。

-オイルバスに敷かれているスターラーのレバーからオンオフを切り替えられる。

-左のダイヤルより回転数を変えられる。

-使用後に浸したフラスコ等のオイル接触部分は必ずヘキサンを用いて拭き取る。
