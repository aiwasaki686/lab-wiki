 # LAHの取り扱い・クエンチ <br>
 ## LAHの秤量までの操作<br>
 -標準装備として白衣、保護めがね、(マスク)を着用
 
 -秤量時はパラピンと薬さじの距離を極力近い位置で試薬を置く。(高い位置から落とすとLAHが舞い散り大変危険)

 ## LAHを用いた反応<br>
-反応性が強く、溶媒と懸濁状態となる場合が多いためLAH懸濁液にもう一方の化合物を滴下することが原則。

-0 °C程度の溶液からdropwiseによる滴下を行うと安定して入れられる。

-場合によっては滴下中に泡がでてくると反応している証左であるため、滴下間隔を調整する。

## LAH反応後のクエンチ操作
-いきなり過剰量の水を加えるのは急激な発熱や火災などの原因となるため厳禁

-まずイソプロピルアルコール(IPA)等の穏やかな溶媒をパスツールなどで1滴ずつ慎重に滴下する(かなり激しい音がする)。

-ある程度反応が落ち着いてきたら段階的にメタノール→水などで様子を見ながら滴下する。

-LAHが白い固体となったときに完全に潰れたことを意味する。

-ロッシェル塩飽和溶液(Potassium Sodium L-(+)-Tartrate Tetrahydrate)はAlを捕捉する性質を持つことから最後に加える。

-分液操作時に過剰なエマルジョンがみられる場合はロッシェル溶液を足すことで解決できることがある。


# ドラフト内で取り扱いが推奨される化合物に関しての取り扱い
-メーヤワイン試薬をはじめとした試薬の中には有毒な気体を生じるものもあるため取り扱いは危険である。このような化合物の秤量は次のようにして行う。

-事前に蓋をした空のバイアルを電子天秤にセットし重さをリセットする。

-空のバイアルをドラフト内に移動し目分量で危険試薬を取る。

-取り入れたバイアルに蓋をして電子天秤にセット危険視薬の重量を量る。

-多い、少ない場合はバイアルをドラフト内へ移動し、再度調整して量り取る。

-反応への添加はドラフト内で行う。

# 反応ででた高濃度の酸処理方法<br>
**酸性廃液の処理について**<br>
-ドラフト中で作業を行う。pH試験紙の様子をみながら飽和NaHCO3水溶液を加える。(一気に入れると吹き出す可能性あり)

-薄緑~黄色あたりになったところで含ハロ廃液(真ん中)に棄てる。

**塩基性廃液の処理について**<br>
-ドラフト中で作業を行う。pH試験紙の様子をみながら飽和NH4Cl水溶液を加える。(一気に入れると吹き出す可能性あり)

-薄緑~黄色あたりになったところで含ハロ廃液(真ん中)に棄てる。

**その他危険試薬について注意点**<br>
-DMF, DMSOなどの皮膚に浸透しやすい溶液を取り扱うときは必ず手袋、保護めがねを装備する。

-試薬、溶媒を入れたときに使用した針で手指を刺した場合血を抜き流し、人を呼び速やかに責任者に報告を行う。気分が悪くなり次第迅速に医療機関へアクセスする。

-気体の発生が予想される反応では反応開始後10~30分程度は風船の大きさを確認し、場合に応じて風船のガス抜きを行う。(爆発、風船破裂の防止)
