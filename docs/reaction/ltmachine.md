# 低温反応機の使い方

>最終更新日: 2026-07-03
>最終編集者: 奥澤敦士

1. 正面左側にあるスイッチ2つをつける(奥に1つと手前に1つ)<br>
<img width="1108" height="1477" alt="S__46104579_0" src="https://github.com/user-attachments/assets/ec177f72-3abb-415a-8da1-fdf5ca9b42d9" /><br>
<img width="1108" height="1477" alt="S__46104580_0" src="https://github.com/user-attachments/assets/66c999c9-abc6-4c83-908f-ff61e34c2e51" /><br>

2. 正面にあるスイッチをONにする<br>

3. 反応機の上のバットに入っているビーズと反応をかけるフラスコを入れる<br>
ビーズはフラスコを十分に覆えるくらい入れたい<br>

4. 正面に表示されている2つの温度について<br>
- 上 -> 反応機内の温度<br>
- 下 -> 設定温度(自分がしたい温度)<br>
温度が表示されているモニターの右側にあるボタンで温度の設定を行う<br>
<img width="1477" height="1108" alt="S__46104581_0" src="https://github.com/user-attachments/assets/a7574693-5163-4145-99d7-454985bffd6e" /><br>

5. 反応終了後、すべてのスイッチを切る。<br>
反応機内のビーズは触れる温度になり次第バットに戻す(戻さないと腐る)<br>
<img width="1108" height="1477" alt="S__46104582_0" src="https://github.com/user-attachments/assets/2869b102-e87b-4911-a1ee-68895b2df7fe" /><br>
