# キャピラリーの作成について<br>
- 持ち物: ピンセット、白衣、保護メガネ<br>
- 場所: 6129教室<br>
1. 黒板前にある器具使用ノートにてガスバーナーを使用する旨を記載する<br>

2. ガスバーナー、マッチ、ガラス管を用意する<br>
ガスバーナーおよびマッチは6129教室前の棚、ガラス管は6130教室の黒板付近にプラスチックのバットに大量に入っている<br>

3. ガスバーナーに火をつけ、青い炎になるよう調節する<br>

4. ガラス管の一端をピンセットでもち、もう一方を手で持って、中央付近を火にあてる<br>

5. ゆっくりガラス管を回しながら温めていき、柔らかくなるまで行う<br>

6. ガラス管がたわむほど温まったら、力一杯ガラス管を左右に引っ張る<br>

7. 伸ばしたガラス管を実験台に置き、手で触れるくらいの温度になるまで冷ます<br>

8. ハートカッター(アンプルカッター)を用いて、適切なサイズに切り分けることで完成!!
