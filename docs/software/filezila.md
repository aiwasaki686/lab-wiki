# FileZilla プロトコル

2026/07 岩崎二颯作成

## 1. 対象

-   Windows 10 / Windows 11
-   管理者権限を有するユーザー

## 2. 使用ソフトウェア

-   FileZilla Client（最新版推奨）

## 3. インストール手順

### 3.1 インストーラーの取得

1.  FileZilla公式サイトへアクセスする。
2.  **「Download FileZilla Client」**をクリックする。
3.  Windows版インストーラーをダウンロードする。

### 3.2 インストール

1.  ダウンロードしたインストーラーをダブルクリックして実行する。
2.  ユーザーアカウント制御（UAC）が表示された場合は **「はい」**
    をクリックする。
3.  ライセンス契約を確認し、**「I Agree」** をクリックする。
4.  インストール対象ユーザーは、必要に応じて選択する。
5.  インストールするコンポーネントはデフォルト設定のままとする。
6.  インストール先フォルダーはデフォルト設定のままとする。
7.  **「Install」** をクリックしてインストールを開始する。
8.  インストールが完了したら、**「Finish」** をクリックする。

## 4. 設定

### 4.1 サイトマネージャーへの登録

1.  FileZilla Clientを起動する。
2.  **「ファイル」→「サイトマネージャー」** を選択する。
3.  **「新しいサイト」** をクリックし、任意のサイト名を入力する。
4.  接続情報を入力する（以下の3つを登録）。

  #### 5119 の NMR
- ホスト：133.91.89.202
- ユーザー：ECZ500
- パスワード：nmr123

#### 6130 の NMR
- ホスト：133.91.89.6
- ユーザー：delta
- パスワード：delta

#### 5119 の mass
- ホスト：133.91.89.63
- ユーザー：AccuTOF
- パスワード：mass


5.  **「接続」** をクリックし、正常に接続できることを確認する。
6.  次回以降は、サイトマネージャーから登録したサイトを選択して
    **「接続」** をクリックするだけで接続できる。
