# 微生物培地作製マニュアル（寒天培地編）

## 概要

寒天培地は、微生物を固体培地上で培養するために使用される培地である。培地に含まれる栄養成分は培養対象によって異なるが、基本的な作製方法は共通している。

本マニュアルでは、本研究室で使用している寒天培地について、用途・培地組成・作製方法・使用上の注意をまとめる。

---

# 培地一覧

| 培地 | 用途 | 対象微生物 |
|------|------|------------|
| R2A寒天 | 一般細菌の単離用 | 細菌 |
| NBRC寒天 | 一般細菌の培養・菌株維持 | 細菌 |
| BG-11寒天 | 無機塩主体の培地 | シアノバクテリア・藻類 |
| ISP-2寒天 | 放線菌の分離・培養 | 放線菌（Streptomyces属など） |
| PDA寒天 | 真菌の培養・菌株維持 | カビ・酵母 |
| GYM寒天 | 放線菌の培養・二次代謝産物生産 | 放線菌 |

---

# 共通操作

## 概要

寒天培地は培地組成こそ異なるものの、基本的な作製方法は共通している。各培地の組成に従って試薬を秤量し、以下の手順で作製する。

---

## 作製手順

1. 三角フラスコへ必要量の精製水を加える。
2. 寒天以外の試薬を秤量し、十分に溶解する。
3. pHを測定し、必要に応じてNaOHまたはHClで調整する。
4. 寒天を加え、均一になるよう十分に混合する。
5. フラスコの口をアルミホイルで覆い、オートクレーブ（120 ℃・20分）で滅菌する。
6. 滅菌後、約50～60 ℃まで冷却する。
7. クリーンベンチ内でシャーレへ分注する。
8. 室温で寒天を固化させる。
9. 必要に応じて培地表面を乾燥させ、パラフィルムで封をして4 ℃で保存する。

---

## 共通注意事項

- pHは寒天を加える前に調整する。
- 滅菌後は50～60 ℃まで冷却してからシャーレへ分注する。
- 分注はクリーンベンチ内で行い、コンタミネーションを防ぐ。
- 培地には作製日、培地名を記載する。
- 作製後は4 ℃で保存し、乾燥を防ぐためパラフィルムで密封する。

# R2A寒天培地

## 概要

R2A（Reasoner's 2A）寒天培地は、栄養要求性が低い環境細菌の分離・培養を目的として開発された低栄養培地である。一般的な栄養培地では生育しにくい水系由来の細菌や環境細菌の培養に適している。

本研究室では、環境試料からの細菌分離および菌株維持を目的として使用する。

### 培地組成（100 mL）

| 試薬 | 添加量 |
|------|-------:|
| 酵母エキス | 0.05 g |
| ポリペプトン | 0.05 g |
| カザミノ酸 | 0.05 g |
| ブドウ糖 | 0.05 g |
| 可溶性デンプン | 0.05 g |
| リン酸水素二カリウム | 0.03 g |
| ピルビン酸ナトリウム | 0.03 g |
| 硫酸マグネシウム七水和物 | 0.005 g |
| アジ化ナトリウム | 0.003 g |
| 寒天 | 1.5 g |
| 精製水 | 100 mL |

**pH：7.2（滅菌前）**

### 特記事項

- 共通操作に従って作製する。
- 低栄養培地のため、生育まで数日～1週間程度かかる場合がある。
- 培養中は培地の乾燥を防ぐ。

# NBRC寒天培地

## 概要

NBRC寒天培地は、一般細菌の培養や保存株の維持に広く用いられる栄養培地である。ペプトンや酵母エキスを含み、多くの環境細菌や非偏性細菌が良好に生育する。

本研究室では、細菌の分離・培養および菌株維持を目的として使用する。

### 培地組成（100 mL）

| 牛肉エキス | 0.1 g |
| N-Zアミン | 0.2 g |
| マルトース一水和物 | 1.0 g |
| 寒天 | 1.5 g |
| 精製水 | 100 mL |

**pH：7.3（滅菌前）**

### 特記事項

- 共通操作に従って作製する。
- 滅菌前にpHを7.3へ調整する。

# BG-11寒天培地

## 概要

BG-11寒天培地は、シアノバクテリアや藻類の分離、単離および維持培養に使用される寒天培地である。液体BG-11培地に寒天を加えて調製し、平板培地として使用する。

本研究室では、シアノバクテリアの単藻化、コロニー形成および保存培養を目的として使用する。

| 試薬 | 添加量 |

| 精製水 | 400 mL |

---

## 作製方法

1. A液を調製し、pHを7.1に調整する。
2. B液は精製水400 mLにBacto Agar 15 gを加え、十分に懸濁する。
3. A液、B液をそれぞれオートクレーブ（120 ℃、20分）で滅菌する。
4. 両液とも約70 ℃まで冷却する。
5. クリーンベンチ内でA液へFe solution（Stock 5）を1.0 mL添加し、よく混合する。
6. A液をB液へ加え、均一になるよう混合する。
7. シャーレへ分注する。
8. 室温で固化させる。
9. 必要に応じて乾燥させた後、パラフィルムで封をし、4 ℃で保存する。

---

## 使用上の注意

- Fe solutionは必ず寒天と混合する直前に添加する。
- A液とB液は70 ℃程度まで冷却してから混合する。
- 寒天は十分に混和してから分注する。
- 分注はクリーンベンチ内で無菌的に行う。
- 保存時は乾燥を防ぐためパラフィルムで密封し、4 ℃で保存する。

# ISP-2寒天培地

## 概要

ISP-2（International Streptomyces Project Medium No.2）寒天培地は、放線菌の培養・菌株維持・形態観察に広く使用される培地である。

本研究室では、放線菌の培養および菌株維持を目的として使用する。

### 培地組成（100 mL）

| 試薬 | 添加量 |
|------|-------:|
| 麦芽エキス | 1.0 g |
| 酵母エキス | 0.4 g |
| ブドウ糖 | 0.4 g |
| 寒天 | 2.0 g |
| 精製水 | 100 mL |

**pH：7.2（滅菌前）**

### 特記事項

- 共通操作に従って作製する。
- ブドウ糖は必要以上に加熱しない。
- 放線菌は培養期間が長いため、培地の乾燥に注意する。

# PDA寒天培地

## 概要

PDA（Potato Dextrose Agar）寒天培地は、ジャガイモ抽出液とブドウ糖を主成分とする培地であり、真菌の培養に広く使用される。

本研究室では、真菌の培養および菌株維持を目的として使用する。

### 培地組成（500 mL）

| 試薬 | 添加量 |
|------|-------:|
| ジャガイモ | 100 g |
| ブドウ糖 | 10 g |
| 寒天 | 10 g |
| 精製水 | 500 mL |

**pH：5.6 ± 0.2（滅菌前）**

### 作製方法

1. ジャガイモ100 gを適当な大きさに切る。
2. 精製水約100 mLで煮沸し、ジャガイモ抽出液を得る。
3. ろ過し、抽出液を三角フラスコへ回収する。
4. ブドウ糖および寒天を加える。
5. 精製水を加えて全量を500 mLに調整する。
6. 必要に応じてpHを確認する。
7. オートクレーブ（120 ℃・20分）で滅菌する。
8. 約50～60 ℃まで冷却後、クリーンベンチ内でシャーレへ分注する。
9. 固化後、パラフィルムで密封し4 ℃で保存する。

### 特記事項

- ジャガイモ抽出液は十分にろ過する。
- 真菌は培養期間が長いため、乾燥を防ぐよう保存する。

# GYM寒天培地

## 概要

GYM（Glucose-Yeast extract-Malt extract）寒天培地は、放線菌の培養や二次代謝産物の生産に適した栄養豊富な培地である。

本研究室では、放線菌の培養および菌株維持、天然物探索に使用する。

### 培地組成（100 mL）

| 試薬 | 添加量 |
|------|-------:|
| 麦芽エキス | 0.4 g |
| 酵母エキス | 1.0 g |
| グルコース | 0.4 g |
| CaCO₃ | 0.2 g |
| 寒天 | 2.0 g |
| 精製水 | 100 mL |

**pH：7.0（滅菌前）**

### 特記事項

- 共通操作に従って作製する。
- CaCO₃は完全には溶解しないため、十分に懸濁させる。
- 必要に応じて超音波処理を行い、寒天を均一に分散させる。
- 放線菌は培養期間が長いため、培地の乾燥に注意する。

# HV寒天培地

## 概要

HV（Humic acid–Vitamin）寒天培地は、腐植酸（Humic acid）を炭素源とし、ビタミンを添加した低栄養培地である。一般細菌の生育を抑えながら放線菌の生育を促進できるため、土壌などの環境試料から放線菌を選択的に分離する際によく使用される。

本研究室では、土壌などの環境試料からの放線菌の分離を目的として使用する。

---

## 培地組成（500 mL）

| 試薬 | 添加量 |
|------|-------:|
| Humic acid | 2 mL |
| Na₂HPO₄ | 100 mg |
| KCl | 340 mg |
| CaCO₃ | 4 mg |
| MgSO₄・7H₂O | 10 mg |
| FeSO₄・7H₂O | 2 mg |
| 精製水 | 500 mL |

### 添加試薬（オートクレーブ後）

| 試薬 | 添加量 |
|------|-------:|
| ビタミンB混合液（100×） | 2 mL |
| シクロヘキシミド | 10 mL |
| ナリジクス酸 | 2 mL |

**pH：7.3～7.4（滅菌前）**

---

## 作製方法

1. 500 mL三角フラスコへ約180 mLの精製水を加える。
2. Humic acid、Na₂HPO₄、KCl、CaCO₃、MgSO₄・7H₂O、FeSO₄・7H₂Oを加える。
3. 超音波処理を行い、十分に分散・溶解させる。
4. 精製水を加えて500 mLにメスアップする。
5. pHを測定し、7.3～7.4となるよう1 M NaOHまたは1 M HClで調整する。
6. 寒天18 gを加える。
7. オートクレーブ（120 ℃、20分）で滅菌する。
8. 約50～60 ℃まで冷却する。
9. クリーンベンチ内でビタミンB混合液、シクロヘキシミド、ナリジクス酸をそれぞれ2 mL添加し、静かに混合する。
10. シャーレへ分注する。
11. 室温で固化後、パラフィルムで密封し4 ℃で保存する。

---

## 使用上の注意

- Humic acidは溶けにくいため、超音波処理を十分に行う。
- pHは寒天を加える前に調整する。
- シクロヘキシミドおよびナリジクス酸は耐熱性が低いため、オートクレーブ後、50～60 ℃程度まで冷却してから添加する。
- 抗生物質は滅菌済みの溶液を使用する。
- 分注はクリーンベンチ内で行い、コンタミネーションを防ぐ。
