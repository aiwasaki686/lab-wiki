# 大型エバポレーター使用方法

>最終更新: 2026-07-02
>最終編集者: 井上楓太

## 基本操作
- はじめに、**湯浴をよく洗う**。このとき、チラーの電源をつけ、「refrigerator」のランプがついていることを確認してから「pomp」を押す。
- 湯浴に水を張り、ポンプをつける。**ポンプの電源はなるべく非常用のコンセントは使わずに、実験台のコンセントを使う。**
- 湯浴の水が大型エバポ全体に流れていることを確認したら湯浴の温度を40 ℃に設定する。
- 大型エバポのコックを閉め、外部の空気が入り込まないようにしてからポンプをつけ、減圧する。(このポンプは大型エバポ内を減圧するものであって、湯浴のものとは別)
- 十分に減圧したらホース元のコックを開き、サンプルを吸引する。
- サンプルはアチアチポイント(図1)が満たされるまで注入し、コックを閉じる。
- 湯浴の温度をMaxにする。(40 ℃のままだと溶媒の気化熱で湯浴の温度が下がり、十分に蒸発しなくなるため)
- 溶媒が飛ぶのを待つ。

## 溶媒の回収方法(図2参照)
- 下の溶媒受けが最大でも3/4程度溜まったら、①のコックを閉じ、上下の溶媒受けの通り道を遮断する。
- **下の溶媒受けのみを昇圧させたいため、③のコックを矢印が書いてある面が下に来るように回す。**(手順があべこべだと逆流するので注意)
- 昇圧したらそのまま、④のコックを開き、ビーカーで溶媒を回収する。(回収した溶媒はなるべく廃液には捨てず、洗浄用のガロン瓶に入れる。)
- 回収したら④→②の順にコックを閉める。
- ③のコックを縦にし、下の溶媒受けを減圧する。
- 下の溶媒受けが十分に減圧したら①、②のコックを開く。
- 再びサンプルを吸引する場合は、装置全体が十分に減圧してから吸引する。

## 濃縮液の回収方法
- **ホース元のコックを開き、昇圧する。**
- **十分に昇圧したらポンプを止める。**(逆流してしまうため順番を間違えないように)
- ナスフラスコを濃縮液回収部分にセットし、コックを開いて回収する。
- さらにサンプルを追加で吸引する場合は、再びコックを閉めて減圧する。

## サンプルを全て濃縮し終わったら
- 濃縮液を回収する。
- 洗浄用メタノールを吸引する。
- メタノールがある程度飛んだら昇圧し、ポンプを止め、ナスフラスコで濃縮液を回収する。
- 濃縮液を回収したら、洗浄用酢酸エチルを吸引する。
- 同様にして濃縮液を回収する。
- 最後に、再び洗浄用メタノールを吸引する。
- 同様にして濃縮液を回収する。
- 湯浴、チラー、湯浴のポンプを止め、電源を切る。
