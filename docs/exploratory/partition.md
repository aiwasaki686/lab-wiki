# 分液操作

2023/5　渡邉作成

## 【用意する物】

- 分液漏斗
- フラスコ（分液漏斗の受け口として使用）
- ガラス漏斗

## 【手順】

1. 分液漏斗のコックが閉じていることを確認してから、漏斗を用いて分液漏斗に溶液を入れる。ガラス栓の切れ込みをずらすようにして栓をする。

2. まず分液漏斗を逆さにし、すぐにコックを開ける（溶液同士が混和することで内圧が高まることがあるため）。ふたたびコックを閉じて分液漏斗を数回縦に振った後、コックを開けガス抜きをする。ガスが出なくなるまでこの操作を行う。  
   ※ガス抜きをするときは、分液漏斗の先を人がいない方向に向ける。人に溶液がかかるのを防ぐため。

3. ガスが出なくなったら分液漏斗を10~15回ほど縦に振る。

4. 元の向きに戻し、栓についたサンプルを洗い込んでから分液漏斗を放置する。  
   ※ガラス栓は抜いた状態にしておく。ガラス栓とすりガラスが固着して抜けなくなるのを防ぐため。

5. 溶液が二層に割れたら、コックを開けて分液する。コックは開けたり閉じたりを繰り返すようにする。コックをずっと開けたまま溶液を流し込むと渦が生じ、上層の溶液が流れ込む可能性がある。

---

※エマルジョンがひどい場合は、

- 翌日まで放置する。
- 水層にNaClを添加（飽和まで）する。  
  と解消する場合が多い。

NaClを添加する場合は、以下の点に注意。

- 水層の分離操作が不充分だと、有機層に多量のNaClが持ち込まれてしまう。
- 水以外の水層の場合は、添加不可（たとえば90%メタノール）。
