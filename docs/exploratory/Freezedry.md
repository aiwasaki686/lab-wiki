# 凍結乾燥機の使用方法

## 使用器具

・凍結乾燥機  
・ナスフラスコ(事前に溶媒を飛ばすフラスコ(バイアル瓶)を凍らせておく。)  
・メタノール(洗浄用)  
・廃液ビーカー  

## 手順

### 開始手順

1. 挟んであるアルミホイルを取り、多岐管のコックが閉まっていることを確認し、多岐管をセットする。

2. 本体横のコックを閉める。

3. 電源を**ON**にする。ここで、ブザーが鳴ったら**setを2秒押す**。

4. **Auto**のランプが点灯していることを確認。

5. **Run/Stop**を押す。

6. 温度が下がった後、圧力が下がり始める。圧力が下がりきってから、凍結乾燥を開始する。(**約30Pa**)

7. フラスコを多岐管に接続し、フラスコ内が十分に減圧されるまでフラスコを抑えながら、コックを**Vac.**(上)にゆっくり回す。  
バイアル瓶で行う場合は、小トラップとアダプターをクリップでつけ、フラスコと同様に行う。  
**また、溶媒を飛ばしている最中に、溶媒が溶けてしまった場合、外してフラスコを冷凍庫で凍らせるか、フラスコの外側に氷をつけて凍らせる。**

### 終了手順

1. 多岐管のコックをゆっくりVent.に回し、フラスコを回収する。

2. Defを長押して、リークを行う。

3. 冷却器が**室温**になるまで待つ。

4. 電源を**OFF**にする。

5. 多岐管を外す。

6. 本体横のコックを開ける。

7. 冷却器をメタノールでよく洗浄する。廃液は本体横のコックから出てくるため、コックの下に廃液ビーカーを用意しておく。

8. 洗浄したら、冷却器と多岐管の間にアルミホイルを挟む。
