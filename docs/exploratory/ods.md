# ODS操作方法

>最終更新:2026-07-02
>最終編集者:井上楓太

## 使用器具

- ナスフラスコ（4～6個）
- クロマトグラフ管
- 三角フラスコ（2個）
- 長いパスツールピペット（2本）
- ビーカー（50～200 mL）
- ガラスろうと
- メスシリンダー
- バイアル瓶
- 薬さじ
- ポンプまたは二連球

## 使用試薬

- Cosmoil 75C18-OPN
- メタノール
- アセトン
- 蒸留水
- クロロホルム

## 操作

### 1. 準備

- ナスフラスコを真空乾燥後、秤量しておく。

-   カラムに綿を詰める。**コック部分に綿が入らないよう注意する。コックが故障する原因となる。**

-   サンプルの10倍量 \[g\] (= 1 vol.)のCosmoil 75C18-OPNをビーカーに秤量し、メタノールに浸して置いておく。

-   次に、コックを開け、メタノールで洗浄する。コックを閉じてメタノール1 cmくらい入れる。

-   ゲル-メタノール懸濁液をガラスろうとを用いて、懸濁させながらカラムに充填する。ポンプを用いてチャージ面ギリギリまで溶媒を流し、コックを閉じる。

-   三角フラスコに40%メタノール溶液 (5 vol.)を2つ作る。（1つは100%メタノールを洗いこむため、1つはサンプルを流す用）

-   コックを閉めたまま、パスツールピペットで40%メタノール5 vol.をカラムに入れる（濃度調製）。このとき、パスツールピペットの先端をカラムの壁面に当ててメタノールを伝わせて入れる。（チャージ面を乱さないようにするため。チャージ面を乱さないようにするため。荒れると試料の流れる速度が変わる。）

-   コックを全開に開ける。チャージ面ギリギリまで溶媒を流し、コックを閉じる。2 mmくらい。

-   冷凍庫からサンプルを取り出して、極微量の100%メタノールで溶かす。1滴ずつ。

-   さっきまで40%メタノールが入っていた三角フラスコに純水を入れ、パスツールピペットで同様にして、カラムに1 cm分くらい入れる。

-   **チャージ面の位置が分かるようにテープを貼っておく**

-   サンプルアプライ用パスツールピペットで、1.2滴バイアル瓶 (小)に入れ、エバポ、真空ライン、秤量し、保存しておく。（活性試験用）

-   廃液用ビーカーから40%メタノール用ナスフラスコに入れ替える。

### 2. サンプルのアプライ

-   サンプルアプライ用パスツールピペットで、サンプルをアプライする。**極微量で！！**

-   サンプル分を2として、2 : 3くらいの水を三角フラスコ同様に入れる。（100%メタノールを40%くらいにするため）ここでコックを開け、線のギリギリor不溶物の上まで落とす。

-   40% MeOHを溶媒用パスツールピペットで元のサンプルのナスフラスコに1吸いくらい入れて、1回だけ洗いこむ。サンプルアプライ用パスツールピペットでカラムに入れる。

-   チャージ面ギリギリまで溶媒を落とす。

-   再度 40%メタノールを少量加え、チャージ面ギリギリまで溶媒を落とす。(この操作を2回行うことで、40%メタノールに溶出する化合物をゲルに完全に吸着させることができる。)

-   三角フラスコから溶媒用パスツールピペットで40%メタノールを入れる。

-   ギリギリまで落とす。この間に次の60%メタノールを1つ作っておく

-   ナスフラスコを入れ替える。

-   60%メタノールを溶媒用パスツールピペットで元のサンプルのナスフラスコに1吸いくらい入れて、1回だけ洗いこむ。サンプルアプライ用パスツールピペットでカラムに入れる。

-   ギリギリまで落とす。

-   三角フラスコから溶媒用パスツールピペットで60%メタノールを入れる。

-   ギリギリまで落とす。この間に次のx%メタノールを1つ作っておく

-   ナスフラスコを入れ替える。

-   x%メタノールを溶媒用パスツールピペットで元のサンプルのナスフラスコに1吸いくらい入れて、1回だけ洗いこむ。サンプルアプライ用パスツールピペットでカラムに入れる。

-   ギリギリまで落とす。

-   三角フラスコから溶媒用パスツールピペットでx%メタノールを入れる。

-   ギリギリまで落とす。この間に次のy%メタノールを1つ作っておく

-   ナスフラスコを入れ替える。

-   95% or 100%メタノールまで同様に行う

-   最後に、クロロホルム / メタノール (1:1) (10 vol.)を溶媒用パスツールピペットで元のサンプルのナスフラスコに1吸いくらい入れて、1回だけ洗いこむ。サンプルアプライ用パスツールピペットでカラムに入れる。

-   それをギリギリまで落とす。

-   三角フラスコから溶媒用パスツールピペットでクロロホルム / メタノール (1:1)を入れる。

- 　溶媒を最後まで流す。

### 3. 濃縮、秤量

-   エバポレーターでナスフラスコ内を濃縮する。

-   濃縮が完了したナスフラスコを真空引きして秤量する。

-   それぞれバイアル瓶に移し替え、エバポ、真空ライン、秤量する。
