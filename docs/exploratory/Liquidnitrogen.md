# 液体窒素

2023/5　齊間作成

## 【手順】

1. 1号館受付で液体窒素汲み取り所のカギを受け取る。（学生のみの場合、二人必要　学生証2枚提示のため）

2. タンクに行き、門のカギを開けて、２つのメーター（右：液面 左：圧力）を確認する。

3. 入口近くに注入口があるのでそこで青いハンドルを回して容器に液体窒素を注ぐ。凍傷防止のために、手袋や保護メガネ着用推奨。手袋は汲み取り所奥のboxの中に入っている。

4. 液体窒素を注ぎ終えたら、チューブが凍っているので気をつけながら元に戻して、再度メーターを確認する。（写真を撮っておくとよい）

5. 1号館受付に鍵を返却し、メーターの値、時間などを記入する。

## 【注意点】

- メーターの針が緑と赤のライン間にあることを確認し、そうでない場合は、くまない。
- 異変を感じたらストップする。
- 液体窒素タンクの下にあるバルブは絶対に触らない。
