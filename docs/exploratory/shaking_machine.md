# 振とう機

振とう培養を行う際に使用する装置である。当研究室には**2台**の振とう機が設置されており、培養目的に応じて使い分ける。

## 1. 恒温振とう機（回転振とう）

- 円を描くように回転する振とう方式。
- 往復振とうと比較して**物理的ストレスが少ない**ため、微生物の培養に適している。
- 温度を一定に保ったまま培養を行うことができる。

## 2. 液体振とう機（往復振とう）

- 左右に往復運動を行う一般的な振とう方式。
- **空気の取り込みが良く**、酸素供給を必要とする培養に適している。
- 往復運動のため、液面が高く上がりやすい点に注意する。

---

# 使用前の確認

振とう機を使用する前に、以下の項目を確認する。

- 電源および振とう開始スイッチがOFFになっていること
- フラスコに栓（綿栓・シリコン栓など）が正しく装着されていること
- 培地量が適切であること
- 装置内に障害物や他者の培養物がないこと

---

# 使用方法

1. 電源および振とう開始スイッチがOFFであることを確認する。
2. 振とうしたい三角フラスコの大きさに合わせて、スプリングを用いて固定位置を調整する。
3. フラスコは**4方向・2段**でしっかり固定する。
4. フラスコを設置し、手で軽く揺らして確実に固定されていることを確認する。
5. 必要に応じて温度を設定する。
6. 装置の電源を入れる。
7. 振とう速度（rpm）を設定する。
8. 振とう開始スイッチを押す。
9. 回転数は徐々に上昇するため、フラスコ内の液があふれないことを確認する。
10. 恒温振とう機では、必要に応じて **Reset** ボタンを押して時間計測を開始する。

---

# 使用後

1. 振とうを停止する。
2. 装置が完全に停止してからフラスコを取り出す。
3. 電源を切る。
4. 装置内に培地などがこぼれている場合は清掃する。
5. スプリングを元の状態に戻し、次の利用者が使用しやすいよう整理する。

---

# 注意

## 恒温振とう機（回転振とう）

- 回転速度は**200 rpm程度**を目安とし、必要以上に高回転にしない。
- チャンバーは回転が完全に停止してから開ける。
- 温度設定を変更した場合は、設定温度になっていることを確認してから培養を開始する。

## 液体振とう機（往復振とう）

- 液量が多すぎると培地があふれる原因となるため注意する。
- 往復運動により液面が高く上がるため、フラスコ容量に対して適切な培地量とする。
- フラスコの固定が不十分なまま運転しない。

---

# トラブル時の対応

- 異常な振動や異音を感じた場合は、直ちに振とうを停止する。
- 培地がこぼれた場合は速やかに清掃し、必要に応じて報告する。
- 装置に異常やエラーが表示された場合は、使うことを避ける。
