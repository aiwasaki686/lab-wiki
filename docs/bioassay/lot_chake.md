# FBSロットチェック

## 概要

Fetal Bovine Serum（FBS）は、細胞培養に不可欠な試薬である。しかし、FBSはロットごとに含有成分が異なるため、細胞の増殖性や生存率に影響を与える場合がある。また、高価な試薬であることから、不適切なロットを購入すると研究や細胞維持に大きな影響を及ぼす。

そのため、大量購入を行う前にロットチェックを実施し、現在使用しているFBSと比較しながら、培養に適したロットを選定する。

---

## 手順

1. メーカーへロットチェック用FBSサンプルの提供を依頼する。
2. 到着したFBSは「**FBS非動化**」プロトコルを参照し、非動化を行う。
3. 以下のFBSを用いてそれぞれ培地を調製する。
   - ロットチェック対象FBS
   - 現在使用しているFBS（コントロール）
   - 過去に増殖不良を示したFBS（ある場合）
4. HeLa細胞およびHL60細胞を各培地で培養し、「**細胞継代**」プロトコルを参照して5回継代する。
5. 各継代において細胞の増殖速度および細胞の状態を比較・評価する。
6. HeLa細胞およびHL60細胞の両方で良好な増殖が確認されたロットを選定し、必要量をまとめて購入する。

---

## 注意事項

- 初回～数回の継代では、それまで使用していたFBSの影響が残るため、**5回程度継代した後に評価する。**
- HL60細胞はFBSロットの影響を受けやすく、初期には良好に増殖していても、継代を重ねると増殖不良を示す場合がある。そのため、最終的な評価は複数回継代後の結果を基に行う。
- ロット間を比較する際は、FBS以外の培養条件（培地組成、播種細胞数、培養条件など）を統一する。
