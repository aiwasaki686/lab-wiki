# アクチン蛍光染色実験（HeLa細胞）

## 前日

1. エタノール消毒したカバーガラスを片面15分間UV殺菌する。
2. 6穴プレートにカバーガラスをピンセットで1枚ずつ入れる。
3. HeLa細胞を播種する。
   - **細胞濃度**：2.0 × 10⁴ cells/mL
   - **播種量**：3 mL/well
4. インキュベーターで培養する。

---

## 当日

### 1. 試薬の準備

- 使用する試薬をすべて室温に戻す。

### 2. 薬剤処理

1. 培地を1.5 mL抜き取る。
2. 薬剤を添加する。
3. インキュベートする。

### 3. 洗浄

1. 培地をすべて除去する。
2. PBS(+)で2回洗浄する。
   - **2 mL/well × 2回**
   - PBSはウェル側面を伝わせながら静かに加える。

### 4. 固定

1. 4% PFA in PBS(-)を加える。
   - **2 mL/well**
   - **室温 10分**
2. PBS(+)で3回洗浄する。
   - **2 mL/well × 3回**

### 5. 染色液の調製（固定中）

遮光して以下を調製する。

- Rhodamine-phalloidin
- Hoechst33258（4 μg/mL）

### 6. 細胞膜透過処理

1. 0.1% Triton X-100を添加する。
   - **1 mL/well**
   - **室温 7分**
2. PBS(+)で3回洗浄する。
   - **2 mL/well × 3回**

### 7. 蛍光染色（以降は遮光）

1. Rhodamine-phalloidin（500 μL/well）を添加する。
2. Hoechst33258（4 μg/mL、500 μL/well）を添加する。
3. 暗所で1時間静置する。

### 8. 洗浄

1. PBS(+)で3回洗浄する。
   - **2 mL/well × 3回**
2. 最後はPBS(+)を1 mL残す。
3. カバーガラスをピンセットで取り出す。

### 9. 封入

1. スライドガラスに60% glycerol/PBS(+)を7 μL滴下する。
2. 細胞が付着している面を下にしてカバーガラスを載せる。
3. スライドガラスとカバーガラスの境目をマニキュアなどで封止する。

### 10. 保管・観察

1. 暗所で保管する。
2. 共焦点レーザー顕微鏡で観察する。




### <a name="clsm">[CLSM使用マニュアル]</a>

# CLSM使用マニュアル（3B113教室）

## 1. 装置の立ち上げ

1. 防塵カバーを取り外す。
2. PCの電源を入れる。
3. 集中電源（①）の電源を入れる。
4. レーザー電源（②）の4つのスイッチ（405、488、561、640 nm）をONにした後、キースイッチをONにする（緑色点灯）。
5. タッチパネルコントローラー（③）背面の電源スイッチをONにする（"Start operation"と表示される）。
6. 水銀ランプ（④、顕微鏡本体下）のON/OFFボタンを押す（"Burner status"が青色点灯）。
7. PCにログインする。
   - **Password:** `olympus`
8. **FV31S-SW**を起動する。
9. **Model: OLYMPUS IX3-SSU**を選択し、**OK**を押す。
   - Cleaningは**No**を選択する。

---

## 2. 目視観察（タッチパネルコントローラー）

> ※基本的には省略可

1. 右上のアイコンを**カメラ**から**目**に変更する。
2. **Observation method**で**DIA**（明視野）を選択する。
   - EPI：蛍光観察
3. **Elements**タブ → **Polarizer**を**IN**にする。
4. 左下のシャッターを開ける。
5. **Brightness**タブから光量を調節する。
6. ピントを合わせる。

### ピントが合わない場合

メニューバーから

**Tool → Window → Microscope**

を開き、

**Z limit setting → Near limit**

を変更する。

- マイクロプレートの場合：**5000 μm**程度

7. ピントが合ったら、右上のアイコンを**目**から**カメラ**へ戻す。

---

## 3. カメラでの観察（ソフトウェア操作）

1. タッチパネルコントローラーで対物レンズ倍率を選択する。
2. **40倍以上**では、**2サンプルごとにオイルを1〜2滴**補充する。
3. 試料をセットする。
   - ホルダーを設置し、その上に**スライドガラスを表面が下になるように置く。**
4. **Ocular**を開き、**LSM Mode**を選択する。
5. **PMT Setting**
   - Mode：**VBF**
6. **Dye & Detector Select**
   - **All Clear**を押す。
7. 使用する蛍光色素を選択する。
   - TD（透過像）は自動設定される。
8. 以下を調整する。

**ポイント**

- 明るさは**LaserよりHVで調整する。**
- 蛍光色素はドラッグして各Phaseへ追加する。
- 顕微鏡右側のハンドルで高さ・XY位置を調整する。
- 推奨値:Laser 5% current 1200~1400

---

## 4. 画像撮影

1. **LSM Imaging → Scan Setting → Scanner**
   - Type：**Galvano**
   - Mode：**Oneway**

2. Scanner設定

|項目|設定|
|---|---|
|Speed|2.0 μs/pixel|
|High voltage correction|ON|

3. **Image Size**

|項目|設定|
|---|---|
|Aspect Ratio|1:1|
|Scan Size|512×512 または 1024×1024|

> ※それ以上では撮影時間が長くなり、蛍光退色（Photobleaching）の原因となる。

4. **Area Setting**
   - 回転
   - デジタルズーム

5. 保存先を指定する。

**Acquire → Normal → フォルダアイコン**

6. **LSM Start**を押して撮影する。

- 指定フォルダへ自動保存される。
- 同じフォルダでは **0001, 0002...** の連番になる。
- 必要に応じて右上から**Scale Bar**を追加する。

### TIFF形式で書き出し

1. 画像を右クリックする。

```
Export
    ↓
Folder
```

2. 保存先を指定する。
3. 明視野を除く場合は**TD**のチェックを外す。
4. 設定

- Save as type：**TIFF**
- ROI overlay：**All ROI**

5. **Save**を押す。

---

## 5. 終了

1. ソフトウェアを終了する。

```
File → Exit → OK
```

2. データをUSBへコピーする。
3. PCをシャットダウンする。
4. タッチパネルコントローラーの左上電源をOFFにし、その後背面スイッチをOFFにする。
5. レーザー電源のキースイッチをOFFにし、その後405・488・561・640 nmの各スイッチをOFFにする。
6. 水銀ランプのON/OFFボタンを押す。
   - **300秒のクールダウン**が始まる。
7. 集中電源（①）をOFFにする。
8. 防塵カバーを装置にかける。
