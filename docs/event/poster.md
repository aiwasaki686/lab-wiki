# ポスター印刷

> 最終更新：2026-07-02  
> 最終編集者：坪井

---

### 1. 大型プリンタを予約する

1. 都心ITセンターのホームページから**オレンジページ**へアクセス。

<a href="https://sites.google.com/g.chuo-u.ac.jp/kitc/printer?authuser=0" target="_blank" rel="noopener">
都心ITセンター（大型プリンタ）
</a>

2. 左上の**三本線メニュー**を開き、**「大型プリンタ」**を選択。

3. **大型プリンタ予約状況カレンダー**で空いている日時を確認。

4. **大型インクジェットプリンタ使用申請書**をダウンロードし、必要事項を記入。

<a href="https://drive.google.com/file/d/12NGKwihsnZVAzyAZ8aNbBud_wBK3SXOP/view" target="_blank" rel="noopener">
大型インクジェットプリンタ使用申請書
</a>

5. **指導教員のサイン**をもらう。

<span style="color:red;">※大型プリンタの予約は、**使用日の3営業日前**までに行く。</span>

### 2. 使用申請書を提出する

記入・署名済みの使用申請書を、**都心ITセンター（3号館4階）**へ提出。


### 3. 印刷前の準備

印刷当日までに、以下を準備。

- 印刷するポスターの**PDFデータ**をUSBメモリに保存する。
- **印刷用紙を持参**する。

<span style="color:red;">※印刷用紙は各自で準備してください。</span>

### 4. 印刷当日

1. 予約時間に都心ITセンターへ行く。
2. 受付で大型プリンタを利用することを伝える。
3. 用紙はスタッフがプリンタへセットしてくれる。
4. USBメモリ内のPDFデータを使用して印刷。
