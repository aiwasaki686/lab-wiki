# マニュアルシム

## 操作手順

1. **「マニュアル測定」** を開く。
2. 変更前の写真を撮っておく
3. gainを上げる。(Lockの値がカンストしないようにする)
4. Z1→Z3→Z2→Z4 の順番でLockの値が大きくなるように調整。±50ずつ動かして、±10で微調整する。
5. gainをLockの値が1000程度になるように調整
6. jobを登録して、測定を開始する。

**note**
- Z1,Z3 → シグナルのブロード化を解消
- Z2,Z4 → シグナルの対称性を解消

# シムの読み込み

## 同じシムで測定したいとき

1. 以前取った NMR を開く
2. 測定画面の左上のシム→シムの読み込み→データファイル
3. 開いておいた NMR をクリック
4. job を登録をして、測定開始する。

## シムファイルの読み込み

1. 測定画面の左上のシム→シムの読み込み→ユーザーシム
2. 溶媒のシムを選択する
3. job を登録をして、測定開始する
