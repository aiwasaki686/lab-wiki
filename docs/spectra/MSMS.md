# MSMS測定・解析

## MSMSの測定準備

1. 中央大学で天然物を0.1 mg 小バイアルに分け取る。
2. 慶応大学で10 μg/mlに希釈する。
3. 測定をする。

## MSMSスペクトルの解析

1. 現在予想しているシーケンスを Chemdraw で作っておく。
2. SeeMSを起動する。
3. 自分のMSMSデータを開く
4. 観測されたフラグメントを過不足なく説明できるようシーケンスをつなぐ。

**note**
- フラグメントもイオン化された状態で観測されている。
- MSMSでは、 [M+Na]⁺ や [M+H]⁺ などのイオン付加体を [M] と記載する。
- 脱水素 [M-H]、水素付加 [M+H]、脱水 [M-H₂O] のフラグメントピークが観測されることがある。
- 脱水素 [M-H] は N 末端側が観測されているとき、水素付加 [M+H] は C 末端側が観測されているときに出やすい。
- 脱水 [M-H₂O] は水酸基をもつ残基 (Serなど) の横で切れたときに起こりやすい。
- ペプチド性の化合物はアミド結合C-N 部分で切れやすい。
- プロリンのアミド結合はより切れやすい性質がある。
