# Chemyx Fusion 100 簡易操作マニュアル

------------------------------------------------------------------------

## 📋 使用前チェック

-   [ ] 電源ケーブル接続
-   [ ] シリンジ・チューブに破損なし
-   [ ] 気泡を除去した
-   [ ] シリンジサイズを確認

> ⚠️ **最重要**
>
> シリンジ設定（容量・内径）が間違っていると、設定流量と実際の流量が一致しない。

------------------------------------------------------------------------

# STEP 1　電源を入れる

1.  背面スイッチをON
2.  ホーム画面が表示されることを確認

> 💡 起動後は初期化が終わるまで待つ。

------------------------------------------------------------------------

# STEP 2　シリンジを取り付ける

<img width="4031" height="3023" alt="Image (1)" src="https://github.com/user-attachments/assets/0f2ac966-5a44-4329-8dde-bebbcf267332" />

1.  リリースボタンを押す
2.  プッシャーブロックを移動
3.  シリンジを固定
4.  プランジャーへ軽く接触させる

> ⚠️ 強く押し付けない。

------------------------------------------------------------------------

# STEP 3　シリンジ設定

<img width="4031" height="3023" alt="Image (2)" src="https://github.com/user-attachments/assets/c587ff69-3e3e-40b2-ab6c-e0e22d2192ef" />

ホーム画面から **Syringe** を選択する。

Find Syringeから

-   メーカー
-   容量

を選択する。

or

Customの場合は

-   内径(id/mm)

を設定する。

> 💡 シリンジ交換後は毎回確認する。

------------------------------------------------------------------------

# STEP 4　送液量 (Volume) を設定

<img width="4031" height="3023" alt="Image (3)" src="https://github.com/user-attachments/assets/d15a2211-3db2-4609-9d49-ba2476653b76" />


例

-   8.0 mL

設定量に達すると自動停止する。

>  **実際の容量より過剰に設定すると、シリンジを必要以上に押し続けて破損に繋がるため注意**

------------------------------------------------------------------------

# STEP 5　流量 (Rate) を設定

<img width="4031" height="3023" alt="Image (4)" src="https://github.com/user-attachments/assets/c1298668-6738-4c04-98a5-1435133335e3" />

例

-   4.0 mL/min

必要に応じて単位を変更する。

Select Unitsから
-   μL/min
-   mL/min
-   μL/hr
-   mL/hr

------------------------------------------------------------------------

# STEP 6　送液開始

<img width="4031" height="3023" alt="Image (5)" src="https://github.com/user-attachments/assets/a7787a5a-5110-4271-ae9f-ffd1407c13ce" />

START を押す。

画面で以下を確認する。


-   現在流量
-   設定送液量
-   経過時間

------------------------------------------------------------------------

# ⏸ 一時停止

PAUSE を押す。

再開するときは START を押す。

------------------------------------------------------------------------

# ⏹ 停止・終了

STOP を押す。

その後

-   シリンジ取り外し
-   液漏れ確認
-   電源OFF

------------------------------------------------------------------------

# 🚨 よくあるトラブル

  表示          原因               対処
  ------------- ------------------ ----------------------------
  Stall         流路閉塞・抵抗大   チューブ確認、流量を下げる
  Flow Error    シリンジ設定ミス   容量・内径を再確認
  Motor Error   過負荷             STOP後に再起動

------------------------------------------------------------------------

# 💡 よくある失敗

-   シリンジ設定を忘れる
-   気泡を抜いていない
-   チューブが折れている
-   プランジャーを強く押す

------------------------------------------------------------------------

# ✅ 使用後チェック

-   [ ] STOPした
-   [ ] シリンジを外した
-   [ ] 液漏れなし
-   [ ] 電源OFF
-   [ ] 周囲を清掃した

------------------------------------------------------------------------

## ⭐ 覚えておくこと

> 💡 **一番大切なのは「シリンジ設定」。**
>
> シリンジを交換したら、容量・内径(ID)を必ず確認する。
